# Course Enrollment System

## Setup

### Backend
1. `cd server`
2. `npm install`
3. Edit `.env` and add your `MONGO_URI`.
4. `npm start`

### Frontend
1. Navigate to `client/` folder.
2. Open `index.html` in Chrome/Edge.

## Features
- JWT Auth
- Role-based Access (Student vs Instructor)
- Date-based Enrollment Restrictions
